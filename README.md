# twitter-blog-clone
Clone of a blogging site with certain features from Twitter

## Installation
- You can install dependencies with `pip install requirements.txt`

## Usage
- cd into `twiddit` and run `python manage.py runserver`
- Go to http://localhost:8000/ to access the site locally.
- You can create a new user, but in order to see posts of other users, you'll have to be following them.
